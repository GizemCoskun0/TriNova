Smart Kitchen Assistant Project
Our project is a smart kitchen assistant designed to take the daily stress out of deciding what to cook. Users can simply enter the ingredients they have at home, and the system will instantly find the best recipes they can prepare.

The application will also include personalized user profiles so it can remember favorite meals, dietary preferences, and food allergies. This allows the system to recommend recipes that match each user’s needs.

In addition, the app will offer custom weekly meal plans based on what the user wants to eat. It will automatically calculate the total ingredient quantities needed for the entire week, compare them with the ingredients already available at home, and generate a smart grocery list for any missing items.
